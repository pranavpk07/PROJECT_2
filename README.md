# 📊 Regional Sales Analysis Dashboard

## 📌 Overview

This Power BI project presents a comprehensive **Regional Sales Analysis** 
across 4 regions — **East, West, North, and South** — with total sales of 
**₹374.30K** and a profit of **₹64.33K** (17.19% margin). The report spans 
5 interactive pages covering sales overviews, regional breakdowns, customer 
insights, and strategic recommendations.

---

## 📄 Report Pages

| Page | Description |
|------|-------------|
| 🏠 Sales Overview | KPIs, product category & region summary charts |
| 🗺️ Regional Analysis | Map view, sales vs profit by region & product table |
| 👥 Customer Insights | Top 5 customers, product-category-region breakdown |
| 💡 Insights & Conclusion | Key findings, analysis & recommendations |
| 📋 All Combined | Unified dashboard with all visuals in one view |

---

## 📊 Key Metrics

| Metric | Value |
|--------|-------|
| Total Sales | ₹374.30K |
| Total Profit | ₹64.33K |
| Profit Margin | 17.19% |
| Top Region | East |
| Weakest Region | West |

---

## ✅ Key Insights

- **East region** leads in both sales and profit across all product categories
- **West region** consistently underperforms and needs targeted improvement
- Business is **over-dependent on top customers** — revenue diversification needed
- **Stationery and Electronics** are the top-performing product categories
- Profit margins (17%) need improvement despite healthy revenue
